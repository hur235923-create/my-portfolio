# MARUNI 리디자인 — UX 케이스 스터디 ① 분석 (Analysis)

**"Minimal in form, deep in soul" / Wood Furniture · Quiet Luxury**
**분석: 허강범 (Heo Gangbeom) · 2026**

이 케이스 스터디는 **① 분석(본 문서) → ② 개선안([2_개선안.md](2_개선안.md)) → ③ 반영([3_반영.md](3_반영.md))** 3부작으로 이어진다.
본 문서는 실제 브랜드 사이트를 분석해 UX 문제(P1~P5)를 도출하는 단계다.

> '기존'은 추정이 아니라 **2026-07-05 실제 maruni.com/en(공식 영문)에 직접 접속·실측한 결과**를 기준으로 한다(→ §3).
> 이 분석에서 도출한 문제를 바탕으로 리디자인 설계(②)와 구현(③)을 정리한다.

## 1. 서비스 이해 (Service Understanding)

· 브랜드 목적
  - 1928년 히로시마에서 시작된 목가구 브랜드. 장인의 기술을 산업 생산으로 옮기는
    'industrialised craft(산업화된 장인정신)'가 정체성.
  - 후카사와 나오토·재스퍼 모리슨과 함께 'Super Normal'(과시하지 않는 일상의 비범함)을 추구.

· 핵심 가치
  - 시간을 초월하는 디자인 · 기계 정밀함 + 손의 온기 · 절제된 아름다움(quiet luxury) · 내구성.

· 주요 사용자
  - (A) 인테리어/가구 애호가: 오래 곁에 둘 '평생 가구'를 신중히 고르는 고관여 구매층.
  - (B) 디자인 리터러시가 높은 층: 디자이너(후카사와/모리슨)·제작 방식에 가치를 두는 사람.
  - (C) 채용담당자/동료: 에디토리얼 감각과 절제된 인터랙션 완성도를 평가.

· 방문 목적
  - 제품의 '스펙'이 아니라 '철학·만듦새·질감'을 음미하고, 소장 가치를 확인.

## 2. 사용자 목표 분석 (User Goals)

· 가장 먼저 찾는 정보
  - "이 브랜드의 태도(철학)" → "대표 컬렉션의 형태와 질감" → "누가, 어떻게 만드는가".

· 주요 행동 흐름
  1) Hero의 대형 우드 공간 이미지로 톤·격 감지
  2) Manifesto에서 'industrialised craft' 철학 이해
  3) Collection에서 Roundish/Lightwood/Hiroshima 가구의 형태·디테일 음미
  4) 마음에 드는 가구를 확대(라이트박스)해 질감 확인
  5) Craft·Designers에서 제작 방식·디자이너 서사로 소장 가치 확신

· 사용자의 진짜 니즈
  - 빠른 구매 전환이 아니라 '느리게 음미하는' 경험. 정보 과잉이 아닌 절제된 큐레이션.
  - 가구 디테일(곡면·결)을 '가까이서' 보고 싶은 욕구.

## 3. 기존 UX 분석 (Baseline UX Diagnosis)

※ 2026-07-05 실제 maruni.com/en(공식 영문)에 접속해 내비·IA·콘텐츠·톤을 실측한 결과다.
  (maruni.com 최상단은 JS 스플래시라 구조가 드러나는 영문 사이트를 기준으로 함. 접근성·모바일 세부는 범위 밖.)

· 정보구조(IA) — 카탈로그·유통 중심
  - 최상위 내비가 PRODUCTS · DESIGNERS · INSPIRATIONS · DEALERS · ABOUT US(+NEWS·PRESS·CONTACT).
    Products는 Search/Catalogs/Product Sheets/Materials, 컬렉션은 컨셉명(Meguro·Shoto·T&O·
    Hiroshima·Lightwood), Dealers는 지역별 대리점 — '제품 탐색·유통' 구조가 앞선다.
  - 브랜드 철학·제작 서사는 About 하위(Philosophy·Factory)로 분리되어 제품 탐색과 단절.

· 홈 구성 — 신규 컬렉션·제품 전면
  - 히어로가 NEW COLLECTIONS 2025(5개 컬렉션 카드)로 시작 → NEWS → PRODUCT SERIES(Hiroshima·T&O)
    → PROJECTS → 뉴스레터/SNS의 '제품·컬렉션 카드 나열' 리듬.

· 헤리티지·디자이너 — 표기는 있으나 서사 아님
  - 창립 '1928'은 홈에서 전면화되지 않음(확인 불가). 디자이너(Fukasawa·Morrison·Nendo·Sanaa 등)는
    'Design by [이름]' 제품 귀속 라벨·개별 페이지로 존재하나, '철학·만듦새'를 잇는 몰입 서사로 엮이지 않음.

· 시각 톤·콘텐츠 — 사진 주도(강점) + 카탈로그 성향
  - 컬렉션별 대형 이미지 카드 중심, 텍스트 최소(3~5줄)로 갤러리형 톤은 선다(강점).
    다만 스토리텔링은 약하고 카탈로그·제품 기능이 앞서, '느리게 음미하는' 서사 밀도는 낮다.

· 인터랙션
  - 대형 카드 클릭·별도 비디오 섹션 위주의 정적 구성. '느린 음미'를 매개하는 확대·전환 리듬 장치는 관측되지 않는다.

· 접근성·모바일 — 이번 실측 범위 밖
  - 렌더 DOM·모바일 전환 세부는 이번 실측에서 확인하지 않았다(추정 배제). 별도 실측 권장.

## 4. 문제 발견 (Pain Points)

P1. 상거래형 IA가 '철학·만듦새'라는 브랜드 핵심을 제품 탐색에서 밀어낸다.
P2. 균질 그리드·가격 강조가 '음미'가 아닌 '비교·스캔' 모드를 유발한다.
P3. 작은 병렬 이미지로는 곡면·나뭇결 같은 소장 가치의 핵심 디테일이 전달되지 않는다.
P4. 스펙 위주 카피로 '왜 이 형태인가/누가 만드는가'의 서사가 약하다.
P5. 확대 시 맥락이 끊겨 '느린 음미'의 리듬이 깨진다.

## 5. 원인 분석 (Why)

P1 ← 전환(구매) 최적화를 우선해 브랜드 서사를 종속 페이지로 분리함.
P2 ← 제품을 '비교 대상'으로 다루고 큐레이션·여백의 위계를 설계하지 않음.
P3 ← 이미지를 썸네일로만 취급, 디테일을 '가까이 보는' 뷰를 제공하지 않음.
P4 ← 콘텐츠를 스펙 데이터로 환원하고 만듦새·디자이너 서사를 승격하지 않음.
P5 ← 확대를 단순 팝업으로 처리해 위치·맥락의 연속성을 고려하지 않음.

## 6. UX 문제 정의 (Problem Statement)

· 사용자는 "오래 곁에 둘 가구의 철학·만듦새·질감을 느리게 음미"하려 하지만,
  상거래형 균질 그리드와 작은 이미지 때문에 '비교·스캔'에 그치고 소장 가치를 확신하지 못한다.
· 사용자는 "가구의 곡면·결을 가까이 보고" 싶지만, 확대 경험이 맥락을 끊어
  음미의 리듬을 방해한다.
