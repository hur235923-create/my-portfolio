## HP1 — 모바일 내비게이션 드로어

· 발견한 문제: ≤720px에서 nav 메뉴를 대체 없이 display:none → 모바일 섹션 이동 수단 전무.
· 개선 목표: 모바일에서 About·Products·Inspiration·Discover 탐색 회복.
· 디자인 해결: 우상단 햄버거 → dark luxury 풀스크린 드로어(Playfair 헤딩 + 부제).
  aria-expanded/controls, 포커스 트랩, ESC, 스크롤 락(hb_lenis), 포커스 복귀.
  reduced-motion 시 트랜지션 제거.
· 기대 효과: 모바일 탐색 경로 회복.
· 검증: 버거 노출·드로어 열림·focusInside=true·ESC 닫힘 통과.

## HP2 — READ MORE 죽은 링크 처리

· 발견한 문제: 다수 "READ MORE"(.link-more)가 href="#" → 클릭 시 페이지 최상단 점프.
· 개선 목표: 오작동 제거, 상세 페이지 부재를 정직하게 처리.
· 디자인 해결: href="#"인 link-more 6개에 aria-disabled="true" + 클릭 무력화(preventDefault).
· 기대 효과: 잘못된 상호작용 제거, 혼란 방지.
· 검증: link-more 6개 중 6개 aria-disabled 적용 확인.

## HP3 — 라이트박스 포커스 트랩 보강

· 발견한 문제: 라이트박스가 열려도 Tab 시 포커스가 뒤 페이지로 이탈(모달 트랩 미비).
· 개선 목표: 모달 열림 동안 포커스를 모달 내부로 가둔다.
· 디자인 해결: open 상태에서 Tab을 닫기 버튼으로 고정.
· 기대 효과: 키보드/스크린리더 모달 조작 일관성.
· 검증: 라이트박스 열림→Tab 시 포커스 닫기 버튼 유지, ESC 닫힘 확인.

## HP4 — 데스크톱 nav 현재 섹션 활성 표시

· 발견한 문제: nav 링크에 현재 섹션 표시가 없어 wayfinding 약함.
· 개선 목표: 현재 위치를 nav에 표시.
· 디자인 해결: 스크롤 시 활성 섹션 링크 aria-current="location" + 밑줄 유지.
· 기대 효과: 데스크톱 wayfinding 향상.
· 검증: 스크롤 시 활성 nav(#discover 등) 정확히 갱신 확인.
