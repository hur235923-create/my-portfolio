# HASSELBLAD 리디자인 — UX 케이스 스터디

**"Legacy in Every Frame" / Heritage Camera · Dark Luxury**  
**분석: 허강범 (Heo Gangbeom) · 2026**  

이 문서는 완성된 리디자인 결과물을 UX 관점에서 "처음부터 다시" 역설계하여,
무엇이 문제였는지 → 왜 문제인지 → 어떻게 해결했는지의 논리로 정리한 것이다.
'기존'은 핫셀블라드류 하이엔드 카메라 브랜드 사이트의 통상적 패턴을 기준으로 한다.

## 1. 서비스 이해 (Service Understanding)

· 브랜드 목적
  - 1941년 창립한 스웨덴 중형 포맷(Medium Format) 카메라 제조사.
  - 모토 "Create to Inspire" — 도구를 파는 것이 아니라 '창작의 유산'을 파는 브랜드.
  - 달 착륙 촬영 등 사진사(史)의 결정적 순간을 함께한 헤리티지가 핵심 자산.

· 핵심 가치
  - 헤리티지(80년) · 극한의 화질(1억~4억 화소) · 장인정신 · 프리미엄(고가·소수 정예).
  - 가격이 아니라 '결과물의 격(格)'으로 설득해야 하는 제품군.

· 주요 사용자 (3층위)
  - (A) 구매 검토 전문가/하이아마추어: X·V·H 시스템·렌즈 스펙과 시스템 확장성을 비교.
  - (B) 브랜드 동경층/입문자: 헤리티지와 작가 사례로 '언젠가의 구매'를 꿈꾸는 잠재고객.
  - (C) 채용담당자/동료(이 포트폴리오 맥락): 리디자인의 완성도와 인터랙션 구현력을 평가.

· 방문 목적
  - 제품 라인업(시스템)의 차이 이해, 실제 촬영 결과물(작례) 확인, 브랜드 신뢰 형성.
  - "이 카메라를 쓰면 나도 저런 이미지를 만들 수 있는가"에 대한 감정적 확신.

## 2. 사용자 목표 분석 (User Goals)

· 가장 먼저 찾는 정보
  - "이 브랜드는 왜 특별한가"(헤리티지) → "어떤 제품군이 있는가"(시스템) → "결과물은 어떤가"(작례).

· 주요 행동 흐름 (Primary Flow)
  1) 히어로에서 브랜드 톤·격 감지 (3초 내 이탈 여부 결정)
  2) About에서 헤리티지 서사로 신뢰 형성
  3) Products에서 X/V/H/렌즈 시스템 구조 파악
  4) Inspiration·Discover에서 실제 작례로 '욕망' 형성
  5) 앰배서더(전문가 신뢰)로 확신 강화 → 다음 행동(문의/제품 상세) 의향 형성

· 사용자의 진짜 니즈
  - 스펙 나열이 아니라 "빛과 질감을 이 정도로 담아낸다"는 시각적 증거.
  - 제품이 많아 보이는 혼란이 아니라 '시스템'이라는 하나의 정돈된 세계관.

## 3. 기존 UX 분석 (Baseline UX Diagnosis)

· 정보구조(IA)
  - 통상 하이엔드 카메라 사이트는 제품군·스펙·지원이 병렬로 나열되어 '카탈로그'처럼 읽힌다.
  - 헤리티지가 별도 하위 페이지로 묻혀 브랜드 서사가 첫 화면의 설득력으로 이어지지 못함.

· 내비게이션
  - 다층 드롭다운·과다한 메뉴로 선택 피로. 프리미엄 브랜드의 '절제'와 상충.

· 시각적 계층
  - 스펙 텍스트와 제품 썸네일이 균질하게 나열되어 '무엇을 먼저 볼지' 위계가 약함.
  - 이미지가 정보의 부속물로 처리되어, 결과물의 감동(핵심 설득 자산)이 축소됨.

· 콘텐츠
  - 기능 중심 카피("~화소, ~센서")가 앞서고, '왜 이 격인가'의 정서적 서사가 부족.

· 인터랙션
  - 정적 스크롤 위주. 카메라 '정밀함/장인정신'이라는 브랜드 속성이 움직임으로 번역되지 않음.

· 접근성
  - 이미지 대체텍스트·포커스 관리·키보드 접근이 화려한 비주얼 뒤로 밀리기 쉬움.

· 모바일 사용성
  - 대형 고해상 이미지의 로딩 부담, 가로형 쇼케이스의 터치 전환 부재.

## 4. 문제 발견 (Pain Points)

P1. 헤리티지가 '읽어야 아는 정보'로 묻혀, 첫 화면에서 브랜드의 격이 즉시 전달되지 않는다.
P2. 제품군이 개별 나열되어 X/V/H가 '하나의 시스템'이라는 구조로 인지되지 않는다.
P3. 결과물(작례)이 부속 이미지로 처리되어 구매 동기의 핵심인 '시각적 증거'가 약하다.
P4. 정적 인터랙션이 '정밀·장인정신'이라는 브랜드 속성을 체감시키지 못한다.
P5. 로딩 순간의 브랜드 경험(첫인상)이 비어 있어, 진입 직후 격이 형성되지 않는다.

## 5. 원인 분석 (Why)

P1 ← IA가 '제품 카탈로그'를 우선하여 브랜드 서사를 종속 페이지로 밀어냄.
P2 ← 제품을 SKU 단위로 다루고 '시스템'이라는 상위 개념을 IA로 승격하지 않음.
P3 ← 이미지를 콘텐츠의 장식으로 간주. 결과물 자체를 '주인공'으로 세우는 뷰가 없음.
P4 ← 모션을 장식으로만 인식. 브랜드 속성(정밀·빛)을 인터랙션 언어로 번역하지 않음.
P5 ← 성능·로딩을 '기다림'으로만 처리하고 브랜드 연출 기회로 활용하지 않음.

## 6. UX 문제 정의 (Problem Statement)

· 사용자는 "이 브랜드가 왜 특별하고, 이 카메라로 어떤 이미지를 만들 수 있는지" 감정적으로
  확신하려 하지만, 헤리티지가 묻혀 있고 결과물이 부속 이미지로 처리되어 확신에 이르지 못한다.
· 전문가는 "제품군의 구조(시스템·렌즈 관계)"를 빠르게 파악하려 하지만, 개별 SKU 나열 탓에
  전체 시스템을 한눈에 조망하기 어렵다.

## 7. 프로젝트 목표 (Goal)

G1. 진입 3초 안에 '헤리티지 · 다크 럭셔리'라는 브랜드 격을 체감시킨다.
G2. 제품을 'X · V · H System'이라는 하나의 정돈된 세계관으로 재구조화한다.
G3. 결과물(작례)을 콘텐츠의 주인공으로 승격해 시각적 설득력을 극대화한다.
G4. '정밀·빛'이라는 브랜드 속성을 인터랙션 언어로 번역하되 60fps·접근성을 지킨다.

## 8. UX 전략 (Strategy)

S1. 서사 우선 IA: Hero → About(헤리티지) → Products(시스템) → Inspiration → Ambassadors
    → Discover(갤러리)의 '감정→구조→증거' 스토리라인으로 단일 페이지를 재편.
S2. 절제된 내비게이션: About/Products/Inspiration/Discover 4개 앵커 + EST.1941 시그니처.
    다층 메뉴를 제거해 프리미엄의 '여백'을 확보.
S3. 제품의 시스템화: 가로 핀(pin) 쇼케이스로 X→V→H→Lens를 '한 축의 흐름'으로 제시,
    번호(01–04)로 시스템 계층을 시각화.
S4. 결과물 주인공화: Inspiration 스토리 + Discover 매트릭스 갤러리 + 라이트박스 확대.
S5. 브랜드 속성의 모션 번역:
    - 시네마틱 프리로더(로딩=막을 올리는 연출)로 첫인상 설계
    - 마우스 반응 WebGL 셰이더 히어로로 '빛'을 체감
    - 헤딩 클립 리빌·키네틱 마퀴로 '정밀·리듬' 표현
    - 커스텀 커서·마그네틱 링크로 정교함의 촉각 은유
S6. 성능·접근성 병행: skip-link, aria 라벨, 대체텍스트, prefers-reduced-motion 고려.

## 9. UI 적용 (Design Solution — 실제 구현 매핑)

· 톤&매너: 블랙 기반 다크 럭셔리 + Playfair Display(세리프)로 헤리티지의 격,
  Noto Sans KR로 본문 가독성 확보 → '고전적 권위 + 현대적 명료함'.
· 프리로더(#preloader): 카운트·바가 차오르며 브랜드명 노출 → 로딩을 '개막'으로 전환(P5 해결).
· 히어로(#hero): 실사 카메라 이미지 + <canvas> WebGL glow, 클립된 2행 타이틀 리빌,
  SCROLL 인디케이터로 다음 행동 유도(G1).
· About(#about): 1941 카운트업 + 대형 세리프 "역사" + 패럴랙스 이미지 그리드로
  헤리티지를 첫 서사 블록으로 승격(P1 해결, G1).
· 키네틱 마퀴: "Create to Inspire — Light defines legacy — Since 1941" 정/역방향 흐름으로
  브랜드 슬로건을 리듬으로 각인.
· Products(#products): 가로 핀 쇼케이스 카드(01 X / 02 V / 03 H / 04 Lens) →
  개별 제품이 아니라 '시스템의 순서'로 읽히게 재구조화(P2 해결, G2).
· Inspiration(#inspiration): 피처 스토리 + 작가·장비 맥락 캡션으로 작례에 서사를 부여(P3).
· Ambassadors: 전문가 인용구·포트레이트로 사회적 증거(신뢰) 제공.
· Discover(#discover): data-span 기반 비대칭 매트릭스 갤러리 + 라이트박스 확대로
  결과물을 주인공으로 전시(P3 해결, G3).
· 접근성: skip-link(#about), 섹션별 aria-labelledby, 의미 있는 alt, 커서/링/마퀴는 aria-hidden.

## 10. 기대 효과 (Expected Outcome)

· 진입 즉시 '다크 럭셔리 헤리티지'가 체감되어 초기 이탈이 줄고 브랜드 격 인지가 상승.
· 제품이 '시스템'으로 조망되어 전문가의 라인업 이해 시간이 단축된다.
· 작례가 주인공으로 전시되어 '나도 저런 이미지를' 동경이 커지고 구매 동기가 강화된다.
· 프리로더·WebGL·클립 리빌이 '정밀·빛'이라는 브랜드 속성을 정서적으로 각인.
· skip-link·aria·alt로 화려함과 접근성을 동시에 만족 → 폭넓은 사용자 포용.

[검증 메모] 본 문서는 콘텐츠·IA·인터랙션 구조를 코드 레벨에서 확인해 작성했다.
레이아웃·모션의 실제 화질/성능(60fps, 모바일 전환)은 브라우저 실측으로 별도 확인 권장.
